Server down: WhatsApp brachte gestern viele Klicks Es kann hin und wieder mal vorkommen, dass Server überlastet sind. Nutzer des Messenger-Dienstes WhatsApp kennen das zur Genüge. Bis auf ein paar böse Facebook-Nachrichten war das bislang kein Problem, doch jetzt hat ja Facebook plötzlich seine Finger im Spiel – und die Medien drehen durch. Egal ob “Süddeutsche”, “Tagesspiegel”, “SPIEGEL ONLINE”, “Welt”, “Huffington Post” oder sonst wer: WhatsApps Server fanden gestern Abend fast überall auf deutschen Nachrichtenportalen statt.
